# Clothing Inventory Management System

A full-stack application for managing clothing inventory with frontend, backend, and database components.

## Project Structure

- **frontend/**: React application for user interface
- **backend/**: Node.js/Express API server
- **database/**: Database schema and configuration

## Features

- View clothing inventory with categories
- Add, edit, and delete items
- Price management
- Category organization

## Tech Stack

- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: SQLite (for simplicity)

## Getting Started

1. Install dependencies in both frontend and backend directories
2. Set up the database
3. Start the backend server
4. Start the frontend development server
